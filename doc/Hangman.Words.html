    Hangman v0.9.5 Hangman.Words

    Module to manage word lists for a given secret length key.

    The need for splitting lists arises when we have arbitrary long word lists/streams, as it provides structure.

    Module internally maintains standardized containers for word lists and keeps track of total word counts. Splits big words list into smaller lists.

    A single Words abstraction can contain a single word list or multiple lists with sizes info, which is represented as a container.

    Words provide extra manageability during storage as lists are binaried leaving a smaller footprint.

    Primary functions are new/2, count/1, add/2, filter/2, and stream/1.
