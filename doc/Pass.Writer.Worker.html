    Hangman v0.9.1 Pass.Writer.Worker

    Module is a GenServer that implements writer worker functionality. Specifically, Pass.Writer.Worker is a write-operation specific module that performs async writes into Pass.Cache ETS table.

    If the Pass.Writer.Worker.write/3 operation fails for whatever reason, it doesn't bring down the table-owning Pass.Cache process and interrupt cache reads. Hence the separation. The primary module method is Pass.Writer.Worker.write/3
