    Hangman v0.9.1 Guess.Action

    Module encapsulates hangman round actions and the data associated with carrying them out.

    Uses function builder strategy to easily be able to add new data retrievers, and updaters

    Two sets of implementation are given. One has a function builder approach the other has a more boilerplate looking function body which is easier to read.

    Written this way for purely erudition purposes
