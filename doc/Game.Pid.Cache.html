    Hangman v0.9.1 Game.Pid.Cache

    Module implements GenServer functionality to provide game server pid caching. Ultimately relies on the gproc module to administer registry facilities.

    Pid caching prevents a game server process from having to be loaded from scratch on every use

    get_server_pid(player_name, secret)

    Checks registry cache for game server pid given unique id, returns cached pid or if not found returns new game pid. Handles race conditions
