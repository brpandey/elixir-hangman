    Hangman v0.9.1 Player.Game

    Module handles game playing for synchronous human and robot player types. Handles relationship between Player.FSM, Game.Server and Player.Events

    Loads up the player specific game components: a dynamic game server, and a dynamic event server

    Manages specific player fsm behaviour (human or robot). Wraps fsm game play into an enumerable for easy running.
