    Hangman v0.9.1 Player.Game

    Module handles relationship between player, game server and player event server

    Supports both synchronous human and synchronous robot types

    Loads up player specific game components : dynamic game server, and event server given player

    Manages specific player fsm behaviour (human or robot).

    Wraps fsm game play into an enumerable for easy running.
