    Hangman v0.9.1 Counter

    Module implements a set of functions for working with letter frequency tallies.

    Conventional Hangman strategy deals exclusively with letter frequency data.

    Counter is a key value store where a key is a letter string and a value is a positive integer.

    Functions

    add_letters(counter, word)

    Adds letters to Counter without checking for duplicate letters. Returns an updated Counter

    add_unique_letters(counter, word)

    Adds letters to Counter. Ensure letters are unique. Returns an updated Counter

    add_words(counter, words)

    Adds words list to Counter. Converts words to char list and then letter counts list. List is then submitted to map for bulk value update. Faster than many tiny map updates

    add_words(counter, words_stream, exclusion_set)

    Routines adds word stream to Counter. Splits word stream into manageable lists, relies on add_words/2 to reduce into Counter
