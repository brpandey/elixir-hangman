    Hangman v0.9.1 Reduction.Engine

    Module implements words reduction engine.

    Reduction load is handled through pool.

    Module distributes reduce requests based on pass key id attribute (name). Reduction workers are started up as part of the reducer pool.

    Pool supervises reduction workers.
