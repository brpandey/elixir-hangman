    Hangman v0.9.5 Hangman.Player.Alert.Handler behaviour

    Module implements alert event handler for Hangman.Events.Manager. Each event is displayed to the user as an alert feed event.

    Alert.Handler is setup per player key and consumes the producer events of the events manager
