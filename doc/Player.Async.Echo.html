    Hangman v0.9.1 Player.Async.Echo behaviour

    Module serves as an asynchronous echo server for the asynchronous robot fsm player type

    Allows the asynchronous robot fsm player to run a game asynchronously without having to be wrapped in a Stream.run, runs automatically itself
