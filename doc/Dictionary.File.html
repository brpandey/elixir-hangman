    Hangman v0.9.1 Dictionary.File

    Provides abstraction for various dictionary files.

    The original dictionary file is transformed into intermediate files as representations of each transformation layer.

    Layers range from unsorted to sorted, sorted to grouped, and grouped to chunked

    Each transform handler encapsulates each layers transform procedure
