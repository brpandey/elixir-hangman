    Hangman v0.9.1 Dictionary.File

    Provides abstraction for various dictionary files.

    The original dictionary file is transformed into intermediate files as cached representations of each transformation layer. Unless the original file changes, transformation every time isn’t necessary since the intermediate files are stored on disk.

    Load time is only determined by the last transformed chunk file, which is optimized for ETS load.

    Transformation types are unsorted to sorted, sorted to grouped, and grouped to chunked. Each transform handler encapsulates each transform procedure
