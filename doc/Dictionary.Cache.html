    Hangman v0.9.1 Dictionary.Cache

    Module loads Dictionary.File words into chunks and stores into ETS. Letter frequency tallies are computed and stored into ETS upon startup. Words identified as random are tagged and stored.

    Implements GenServer.

    Provides lookup routines to access chunks, tallys, and random words

    lookup(mode :: atom, pos_integer) ::
  Chunks.t |
  Counter.t |
  [String.t] |
  no_return

    Cache lookup routines

    The allowed modes:

    • :random - extracts count number of random hangman words.
    • :tally - retrieve letter tally associated with word length key
    • :chunk - retrieve the word data chunk associated with the word length key
